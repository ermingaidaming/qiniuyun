# Demo 视频脚本

建议控制在 3 到 5 分钟，必须有声音讲解。

## 1. 项目开场

- 说明项目名称：SpeakFlow Coach。
- 说明选题方向：实时 AI 英语口语陪练。
- 说明目标用户：需要即时口语练习反馈的学习者。

## 2. 运行方式

- 展示 README 中的本地启动命令。
- 启动后端服务。
- 打开浏览器访问本地页面。
- 展示 API 健康状态。

## 3. 核心功能演示

- 选择练习场景。
- 开启麦克风，展示音量电平和 VAD 状态。
- 输入或口述一句英文回答。
- 提交后展示 AI 回复、纠错建议、延迟、CEFR 估计。

## 4. 技术架构说明

- 前端：浏览器麦克风、Web Audio API、实时练习界面。
- 后端：会话 API、mock 反馈逻辑。
- 后续真实链路：WebRTC、VAD、STT、LLM、TTS、抢话打断。
